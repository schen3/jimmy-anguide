Directives

There are three kinds of directives in Angular:

  • Components—directives with a template.
  • Structural directives—change the DOM layout by adding and removing DOM elements.
  • Attribute directives—change the appearance or behavior of an element, component, or another directive.
You can apply many attribute directives to one host element BUT you can apply only one structural directive to a host element.

Structural Directive

Structural directives are easy to recognize. An asterisk (*) precedes the directive attribute name *ngIf, *ngFor

The asterisk is "syntactic sugar". Internally, Angular translates the *ngIf attribute into a {{' element, wrapped around the host element.
