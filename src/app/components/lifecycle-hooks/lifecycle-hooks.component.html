Lifecycle Hooks

What is a lifecycle hook in Angular?
Angular lifecycle hooks are methods defined by some interfaces in the app so that the user implements specific code at key points in the lifecycle of the application. Angular does not call every hook but only the ones that are defined in the code. The interfaces are optional but is a good practice to implement them to get the benefits of strong typing in TypeScript.
