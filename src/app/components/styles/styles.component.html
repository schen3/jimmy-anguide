What are the encapsulation views for a component available in Angular?

  • ShadowDom - The component uses the browser's native ShadowDom implemtation. Component as well as styles are placed withing the shadow DOM

  • Native - Uses a now deprecated version of the browser's native shadow DOM implementation

  • Emulated - This is the default implementation. This emulates the native ShadowDom by renaming the styles to target the component in question

  • None - Angular does not view encapsulation. The styles are added to the global styles 

// None encapsulation
import {{'{'}} ViewEncapsulation } from '@angular/core';

@Component({{'{'}}
  selector: 'my-component',
  templateUrl: 'my-component.component.html',
  styles: [...],
  encapsulation: ViewEncapsulation.None
})
